Italian Localization - FatturaPA - Emission / Fattura Elettronica + FatturaPA

This module allows you to generate the fatturaPA XML file version 1.2 http://www.fatturapa.gov.it/export/fatturazione/it/normativa/norme.htm to be sent to the Exchange System

http://www.fatturapa.gov.it/export/fatturazione/it/sdi.html

Questo modulo permette di generare il file xml della fatturaPA versione 1.2 http://www.fatturapa.gov.it/export/fatturazione/it/normativa/norme.htm per essere spedita al sistema di interscambio SDI http://www.fatturapa.gov.it/export/fatturazione/it/sdi.htm

Leggere attentamente le informazioni di compatibilità e installazione tecnica disponibili nel modulo l10n_it_einvoice_base.

Per maggiori informazioni visitare il sito www.odoo-italia.org o contattare l'ultimo autore: Antonio M. Vigliotti <antoniomaria.vigliotti@gmail.com>.
